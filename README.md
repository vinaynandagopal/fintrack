# FinTrack - Personal Finance Tracker

FinTrack is a fintech web application that helps users manage personal finances.

## Features

- User registration
- User login
- Add income
- Add expenses
- Categorize transactions
- Set monthly budgets
- View dashboards
- Track savings
- Generate reports

## Tech Stack

- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: Python Flask
- Database: MySQL
- Version Control: Git and GitHub
- DevOps: Docker, Jenkins, Kubernetes
- Cloud: AWS EC2
